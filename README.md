# Proyecto_final
# Cotizador Automático TK-U

## Funcionamiento
El programa solicita información del asegurado: edad, estado civil, edad del cónyuge (si aplica) y cantidad de hijos.  
Se calcula el precio total del seguro a partir de un plan base de Q.2,000 y se aplican recargos según las condiciones:

- Recargo por edad del asegurado.
- Recargo por edad del cónyuge.
- Recargo por cada hijo.

El resultado se muestra en pantalla.

## Partes importantes del problema
- Validar que el asegurado sea mayor de edad.
- Aplicar recargos según rangos de edad.
- Considerar el estado civil y la edad del cónyuge.
- Calcular recargos por hijos.

## Posibles mejoras
- Permitir ingresar diferentes precios base según el tipo de plan.
- Guardar las cotizaciones en un archivo o base de datos.
- Validar entradas para evitar errores (ej. texto en lugar de números).
- Crear una interfaz gráfica para hacerlo más amigable.
